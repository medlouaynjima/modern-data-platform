# Phase 9: Serving Layer

## Overview

Phase 9 exposes the curated Gold data to downstream consumers and business users. It consists of a REST API built with **FastAPI** to serve the data programmatically, and a Business Intelligence dashboard built with **Streamlit** to visualize the data interactively.

## Architecture

The serving layer bridges the gap between the storage engine and the end-user:

1. **Spark Thrift Server**: Acts as the SQL engine over our Delta Lake. It serves the Gold Delta tables (`fct_daily_sales`, `fct_customer_activity`, `fct_inventory_position`) located in `/opt/spark/work-dir/data/gold`.
2. **FastAPI**: A lightweight, high-performance Python web framework. It connects to the Gold Delta Lake files natively and exposes the data via RESTful JSON endpoints.
3. **Streamlit**: A Python-based dashboarding tool that fetches data from the FastAPI backend and renders it using interactive Plotly charts and KPI metrics.

## Implementation Details

### 1. FastAPI Backend
The API service (`fastapi/main.py`) exposes three primary endpoints:
- `GET /sales/daily`: Aggregates the last 30 days of gross revenue and order counts from `fct_daily_sales`.
- `GET /customers/top`: Identifies the top 10 customers by lifetime revenue from `fct_customer_activity`.
- `GET /inventory/position`: Retrieves the latest 50 snapshot records from `fct_inventory_position`.

*Note on Data Access*: Due to the transient nature of the embedded Derby metastore in the Spark Thrift server container, FastAPI is configured to query the Delta tables natively by path (e.g., `SELECT * FROM delta.\`/opt/spark/work-dir/data/gold/fct_daily_sales\``) to ensure robustness across container restarts.

### 2. Streamlit Dashboard
The BI dashboard (`streamlit/app.py`) is designed with a sleek, dark-mode aesthetic. It features:
- **Top KPI Cards**: Displaying Total Revenue, Total Orders, and the Top Customer ID.
- **Sales & Revenue Tab**: An interactive line chart tracking daily revenue trends.
- **Customer Activity Tab**: A bar chart highlighting the revenue generated by the top 10 customers.
- **Inventory Tab**: A raw data view of the latest inventory snapshots.

## Usage

To spin up the serving layer:
```bash
docker compose --profile serving up -d
```

- **FastAPI Swagger UI**: Accessible at [http://localhost:8000/docs](http://localhost:8000/docs). Here you can test the endpoints interactively.
- **Streamlit BI Dashboard**: Accessible at [http://localhost:8501](http://localhost:8501).
